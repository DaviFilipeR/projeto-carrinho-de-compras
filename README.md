<h1 align="center"> Projeto Carrinho de Compras </h1>

O projeto carrinho de compras é um projeto disponibilizado no curso de lógica de programação da Alura que simula o funcionamento inicial de um carrinho de compras digital. Nele é possível adicionar itens ao seu carrinho com seus devidos preços e será calculado automaticamente o valor total a ser pago.
